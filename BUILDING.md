# Building ControllerOverlayPro

## Requirements
- Visual Studio 2022
- BakkesMod PluginSDK installed

## Build
1. Open ControllerOverlayPro.sln
2. Set Configuration = Release
3. Set Platform = x64
4. Build Solution

## Notes
- Plugin is client-side only (visual overlay).
- Skins are generic and contain no official logos/symbols.
