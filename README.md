# Controller Overlay Pro

Controller Overlay Pro is a BakkesMod plugin for Rocket League that displays a real-time
controller input overlay on screen.

The plugin is designed for streamers and players who want a visual representation
of their controller inputs while playing.

## Features
- Real-time controller input overlay
- Xbox, PS4, and PS5-style skins
- Adjustable position and scale
- Configurable toggle hotkey (default F9)
- Overlay remains visible while BakkesMod menu is open
- Client-side only (visual overlay)

## Installation (Users)
1. Download the install ZIP
2. Extract into: (Where ever BakkesMod folder is installed) %AppData%\bakkesmod\bakkesmod\
3. Launch Rocket League
4. Open BakkesMod (F2) → Plugins → Controller Overlay Pro
